# Muza Landing — PRODUCT.md

## What

Публичный лендинг muza.lol для десктоп-приложения Muza — бесплатного музыкального плеера для Windows с полными синхронизированными текстами песен без цензуры. Цель страницы: убедить посетителя скачать приложение (кнопка → GitHub Releases).

## Register

brand — лендинг, дизайн и есть продукт. Концепция «Живой плеер»: герой — живой DOM-мокап окна приложения, страница сама говорит на языке дизайна Muza.

## Platform

web (статика: Next.js `output: 'export'`, деплой на muza_2 через Caddy).

## Voice

Русский, на «ты», sentence case, без восклицаний и эмодзи. Коротко, спокойно, по делу — «строго, но живо». Три слова голоса: тёплый, ночной, уверенный.

## Identity (committed — не изобретать заново)

Источник истины: `../muza-design-system/project/` (токены скопированы в `src/app/globals.css`).

- Фон: тёплый почти-чёрный `#121110` / `#171614`. Тёмная тема — единственная (продукт ночной, музыкальный).
- Один акцент `#3b82f6` (темы: red `#f76967`, bolt `#327ad9`) — ≤10% поверхности, только действия и выделение.
- Слои вместо рамок: `--surface-1..4` (прозрачный белый). ЗАПРЕЩЕНО: градиенты, box-shadow/text-shadow, свечения, border-разделители, чистые #fff/#000.
- Шрифты: Golos Text (весь UI) + Unbounded (только бренд-моменты). Иконки lucide, stroke 1.75.
- Motion: 150–250 мс ease-out (вход 400 мс, сдвиг 10px + фейд), без bounce; `prefers-reduced-motion` обязателен.
- Логотип: `public/logo/glyph.svg` — не перерисовывать.
